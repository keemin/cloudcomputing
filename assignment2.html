Pre-requisite requirement

  • Action.io is used as deployment tool.
  • Ensure that GAE is installed
Step1: Create a folder in Action.io Step2: Create a configuation file: app.yaml Step 3: Create a file:main.py Step 4: Type import webapp2,urllib2,csv in main.py Step 5: Create def get(self) function to print out your results Step 6: Use urllib2.urlopen(/url) to fetch the csv file under the URL link. Step 7: Create def mapfunction(data) Step 8: Create def shuffle(mapfunction,data) Step 9: Create def reducerfunction(key,value) Step 10: Use self.response.write() to print out the result
